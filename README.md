# ☁️ Weather Image Classification using CNN

> **Academic Context:** Developed as a core project during my Artificial Intelligence undergraduate studies at the University of Hail, demonstrating the practical application of Deep Learning in computer vision tasks.

## 🌟 Project Overview
This project focuses on **Computer Vision**, where I built a Deep Learning model to automatically classify weather images into different categories (Cloudy, Rainy, Shiny, Sunrise). 

## 🧠 Model Architecture
I utilized a **Convolutional Neural Network (CNN)** to extract features from images. CNNs are highly effective for image recognition tasks because they can capture spatial patterns like edges and textures.

## 🛠️ Technologies Used
* **Frameworks:** TensorFlow & Keras
* **Libraries:** Python, NumPy, OpenCV, Matplotlib
* **Environment:** Google Colab

## 📈 Key Achievements
* **Automated Feature Extraction:** Used Convolutional layers to process raw pixels.
* **Accuracy:** Achieved high performance on the testing dataset.
* **Image Preprocessing:** Implemented resizing and normalization to prepare images for the neural network.

## 🖼️ Dataset
The model was trained on a diverse dataset of weather images to ensure robustness in different lighting and environmental conditions.
